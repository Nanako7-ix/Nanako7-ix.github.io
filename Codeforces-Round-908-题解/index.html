A - Secret Sport

Tag

暴力

题意

一场比赛分成大回合和小回合。在每个大回合中，优先打赢 xxx 场小回合的人赢得这场大回合，优先在 yyy 个大回合中获胜的人将赢得比赛。有人赢得了大回合的胜利，那么这个大回合结束。如果有人赢得了比赛的胜利，同样也不会继续比赛。现在 xxx 和 yyy 是未知的。给你一个由 AB 组成的字符串，表示两个人在小回合的获胜情况。问你能否确定这场比赛一定是谁获胜。一定的意思是，不存在其他的 (x,y)(x, y)(x,y) 组合使得对方获胜。如果可以确定，输出比赛的胜利者 ‘A’ 或 ‘B’，否则输出 ‘?’。

字符串长度 n≤20n \leq 20n≤20，测试组数 t≤104t \leq 10^4t≤104。

分析

注意到 x,y≤nx, y \leq nx,y≤n 否则这把游戏不能结束。那么枚举 (x,y)(x, y)(x,y)，然后再暴力 check 在这种 (x,y)(x, y)(x,y) 配置下谁能获胜或是这种配置不正确。对每种配置的结果统计一下即可。

B - Two Out of Three

Tag

构造

题意

给你一个长度为 nnn 的数组 aaa，构造一个长度为 nnn 的由 123123123 构成的数组 bbb，使得下面的 333 句话只有两句成立。n≤100n\leq 100n≤100，ai≤100a_i\leq 100ai​≤100。

  1. 存在 1≤i,j≤n1\leq i, j \leq n1≤i,j≤n 满足 ai=aja_i = a_jai​=aj​ 且 bi=1,bj=2b_i = 1, b_j = 2bi​=1,bj​=2
  2. 存在 1≤i,j≤n1\leq i, j \leq n1≤i,j≤n 满足 ai=aja_i = a_jai​=aj​ 且 bi=1,bj=3b_i = 1, b_j = 3bi​=1,bj​=3
  3. 存在 1≤i,j≤n1\leq i, j \leq n1≤i,j≤n 满足 ai=aja_i = a_jai​=aj​ 且 bi=2,bj=3b_i = 2, b_j = 3bi​=2,bj​=3

分析

发现只有相同的数字会影响上面 333 句话的成立与否，那么不妨按照数字的分类来做。考虑一种数字，它们对应的 bbb 的值不能能出现 123123123 中的 333 种，否则会满足上面的 333 句话。如果只包含了一种，那么上面的 333 句话都不会成立。

所以对于只出现了 111 次的数字，可以随便给它安排一个 bbb 值。对于出现了超过 222 次的数字，你可以给他们安排 222 个数字，然后它们将让上面的一句话成立。这样就有构造方案了。

D - Neutral Tonality

Tag

平衡树

题意

给你一个长度为 nnn 的数组 aaa，你需要在 aaa 中插入 mmm 个数字，使得最终产生的数组的 LIS 最小。n,m≤2⋅105n, m \leq 2\cdot 10^5n,m≤2⋅105。

分析

约定一个元素的贡献为以该元素结尾的 LIS 的长度。

注意到原来 aaa 中的元素的顺序是不会变的，也就是它们产生的贡献是不会减小的。那么我们在插入 xxx 需要考虑的是能否让它们的贡献不增，同时不产生大的贡献。

不让原来的数的贡献增大其实比较简单，只要保证 xxx 的后面不存在比 xxx 大的数即可。那么可以贪心地想到：我从右往左找最后一个不比 xxx 大的数，然后把 xxx 插在它的前面。

在这种情况下考虑以下是否会产生大的贡献，我们不妨记 xxx 插入的位置为 ppp，如果 p=1p = 1p=1 则显然不会产生大的贡献。根据找插入位置的方法可以发现在 ap−1>xa_{p - 1} > xap−1​>x，它是一定不会在 xxx 的 LIS 中的，也就是只需要考虑区间 [1,p−2][1, p - 2][1,p−2]。那么你会发现 xxx 的贡献是一定不会大于 apa_pap​ 的贡献的，因为同样是考虑区间 [1,p−2][1, p - 2][1,p−2] 而 ap−1>xa_{p-1} > xap−1​>x。

这就证明了在这种构造方式下，数组最大的 LIS 是不会改变的。

接下来如何实现这个插入和查找，只需要使用平衡树上二分即可，没什么难度。
