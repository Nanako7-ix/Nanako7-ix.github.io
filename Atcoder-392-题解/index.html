分析

假设 f(i)f(i)f(i) 为戴着数字 iii 围兜的人所盯着的人的围兜上写着的数字，那么对于 iii 有：

f(q(i))=q(p(i)) f(q(i)) = q(p(i)) f(q(i))=q(p(i))

题意

有 nnn 枚骰子。其中 iii 个骰子有 kik_iki​ 个面，上面写着数字 ai1,ai2,…,aikia_{i1}, a_{i2}, \ldots, a_{ik_i}ai1​,ai2​,…,aiki​​ 。当你掷这个骰子时，每个面出现的概率是 1ki\frac{1}{k_i}ki​1​ 。

你从 nnn 个骰子中选择两个骰子并掷出。请计算在最佳选择骰子的情况下，两颗骰子出现相同数字的最大概率。

其中数据范围是：2≤n≤1002 \leq n \leq 1002≤n≤100, ∑k≤105\sum k \leq 10^5∑k≤105, aij≤105a_{ij} \leq 10^5aij​≤105。

分析

考虑最暴力的解法，枚举骰子的组合，一共有 n2n^2n2 种。然后计算答案也是暴力计算值域中每一个数同时出现的概率。

这个虽然听起来很暴力，算一下也是 10910^9109 次操作感觉很容易被卡，但实际上跑得飞快。

E - Cables and Servers

Tag

并查集、模拟

题意

给你 nnn 个点 mmm 条边的图（2≤n≤2⋅105,n−1≤m≤2⋅1052\leq n\leq 2\cdot 10^5, n - 1 \leq m \leq 2\cdot 10^52≤n≤2⋅105,n−1≤m≤2⋅105）。你可以执行操作：选择一条边，将修改它的一个顶点。问至少需要多少次操作，可以让图连通。输出操作次数和操作方案。保证操作一定存在。

分析

先考虑操作次数，现在你有 kkk 个连通块，显然至少一个连通块存在多余的边，那么可以经过一次操作连接到另外的连通块，这样就变成了 k−1k - 1k−1 个连通块。所以容易发现最终需要执行操作的次数就是 k−1k - 1k−1。

如何输出方案。既然我需要找到多余的边，那么我只需要在存图的时候将多余的边储存起来即可，到时候再用即可。整个过程需要使用到并查集，这个比较显然吧，因为需要维护连通性。

G - Fine Triplets

Tag

多项式乘法/卷积

题意

给你一个数组，数组内的数字不重复且值域为 10610^6106。你需要求出满足 a<b<ca < b < ca<b<c 且 b−a=c−bb - a = c - bb−a=c−b 的 (a,b,c)(a, b, c)(a,b,c) 三元组的个数。

分析

先写暴力，枚举两个数，查找第三个数。

其实这个 f(a)⋅f(2b−a)f(a) \cdot f(2b-a)f(a)⋅f(2b−a) 就已经非常卷积的式子了，所以你会发现实际上就是只需要求 f2(x)f^2(x)f2(x) 这个多项式。

求出来的这个 f2(x)f^2(x)f2(x) 的含义是：有多少对 (a,b)(a, b)(a,b) 满足 a+b=xa + b = xa+b=x。这个时候你只需要枚举 bbb，然后 a+c=2ba + c = 2ba+c=2b 的 (a,c)(a, c)(a,c) 数量为 f2(b)−12\frac{f^2(b) - 1}{2}2f2(b)−1​，减 111 是去掉 a=b=ca = b = ca=b=c 的情况，除以 222 是去除 (a,c)(a, c)(a,c) 和 (c,a)(c, a)(c,a) 的重复。

求 f2(x)f^2(x)f2(x) 的话就是使用快速傅里叶变换（fft）或者快速数论变换（ntt）即可
