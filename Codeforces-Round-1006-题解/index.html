分析

注意到 xxx 的选取范围是 ∣p∣|p|∣p∣ 之内的，所以 kkk 只需要考虑它的绝对值。然后直接贪心地操作，将 aia_iai​ 设为 ppp，一共需要 ⌈kp⌉\lceil\frac{k}{p}\rceil⌈pk​⌉ 次操作。

C - Creating Keys for StORages Has Become My Main Skill

Tag

贪心，构造

题意

给你两个整数 n,xn, xn,x，你需要构造一个长度为 nnn 的序列，使得序列的 a1∣a2∣⋯∣an=xa_1 | a_2 | \cdots | a_n = xa1​∣a2​∣⋯∣an​=x 且 mex⁡{a1,a2,⋯ ,an}\operatorname{mex}\{a_1, a_2, \cdots, a_n\}mex{a1​,a2​,⋯,an​} 最大。数据范围 ∑∣si∣≤2⋅105\sum |s_i| \leq 2\cdot 10^5∑∣si​∣≤2⋅105。

分析

序列的按位或和为 xxx 是一个约束条件，我们先不管他，考虑让 mex⁡\operatorname{mex}mex 最大。这个时候数组是 0,1,⋯ ,n−10, 1, \cdots, n - 10,1,⋯,n−1，而且你是优先放前面的数字，所以你就从 000 开始到 n−1n - 1n−1 把数字放进 aaa 中。然后现在考虑这个约束条件，如果存在一个 aia_iai​ 满足二进制上某一位为 111，那么整个数组的按位或和的这一位上就是 111，而 xxx 在这位上为 000 那就似了。也就是不能出现某一位上 aia_iai​ 是 111 而 xxx 是 111 的情况，这种条件使用 ai∣x=xa_i | x = xai​∣x=x 来判即可。接下来你放的数全是合法的，那么还有一个条件是和为 xxx。这个很简单，如果和不为 xxx，你直接把最后一位换成 xxx 就行了。

G - I’ve Been Flipping Numbers for 300 Years and Calculated the Sum

Tag

数论分块

题意

定义 rev⁡(n,p)\operatorname{rev}(n, p)rev(n,p) 为: 将十进制数 nnn 写成 ppp 进制数，即 n=nt−1…n1n0‾n = \overline{n_{t - 1} \ldots n_1 n_0}n=nt−1​…n1​n0​​。然后翻转 nnn，即 m=n0n1…nt−1‾m = \overline{n_0 n_1 \ldots n_{t - 1}}m=n0​n1​…nt−1​​。mmm 的值即为 rev⁡(n,p)\operatorname{rev}(n, p)rev(n,p)。例如 rev⁡(23,11)\operatorname{rev}(23, 11)rev(23,11) 就是先写成 111111 进制: (21)11(21)_{11}(21)11​ 然后翻转为 (12)11(12)_{11}(12)11​。最后 rev⁡(23,11)=(12)11=13\operatorname{rev}(23, 11) = (12)_{11} = 13rev(23,11)=(12)11​=13。

给你两个整数 n,pn, pn,p，计算 ∑p=2krev⁡(n,p)\sum\limits_{p = 2}^{k} \operatorname{rev}(n, p)p=2∑k​rev(n,p)，数据范围 n≤3⋅105,k≤1018n\leq3\cdot 10^5, k\leq 10^{18}n≤3⋅105,k≤1018。注意不保证 ∑n\sum n∑n 的值，也就是 ∑n\sum n∑n 可以达到 2⋅1092\cdot 10^92⋅109 的数据范围。

分析

注意到当 p>np > np>n 时，nnn 的 ppp 进制数的长度为 111，此时 rev⁡(n,p)=n\operatorname{rev}(n, p) = nrev(n,p)=n。这样我们就只需要计算 [2,min⁡(n,k)][2, \min(n, k)][2,min(n,k)] 范围内的 rev⁡(n,p)\operatorname{rev}(n, p)rev(n,p) 和。

注意到只有当 p2≤np^2 \leq np2≤n 的时候，nnn 的 ppp 进制数的长度大于等于 333，这个时候我们暴力计算 rev⁡(n,p)\operatorname{rev}(n, p)rev(n,p)，时间复杂度 O(nlog⁡n)O(\sqrt{n} \log n)O(n​logn)。

那么剩下的部分都是nnn 的 ppp 进制数的长度为 222 的情况了，那么我们记 n=tx‾n = \overline{tx}n=tx。考虑按 ttt 分类讨论，那么我们可以确定 ppp 的取值范围是: nt+1<p≤nt\frac{n}{t + 1} < p \leq \frac{n}{t}t+1n​<p≤tn​，接着操作一下就会变成 ⌊np⌋=t\lfloor\frac{n}{p}\rfloor = t⌊pn​⌋=t。如果我们能够对块内的任意一个子区间 [l,r][l, r][l,r] 都能计算 ∑p=lrrev⁡(n,p)\sum\limits_{p = l}^{r} \operatorname{rev}(n, p)p=l∑r​rev(n,p)，那么我们可以使用数论分块。

我们考虑 n=tx‾n = \overline{tx}n=tx，进制数为 ppp。首先 t=⌊np⌋t = \lfloor\frac{n}{p}\rfloort=⌊pn​⌋，并且有 x=n−tpx = n - tpx=n−tp。翻转后的数 m=xt‾=(n−tp)⋅p+t=np−tp2+tm = \overline{xt} = (n - tp) \cdot p + t = np - tp^2 + tm=xt=(n−tp)⋅p+t=np−tp2+t。所以 ∑p=lrrev⁡(n,p)=∑p=lrnp−tp2+t\sum\limits_{p = l}^{r} \operatorname{rev}(n, p) = \sum\limits_{p = l}^{r} np - tp^2 + tp=l∑r​rev(n,p)=p=l∑r​np−tp2+t。比较简单的式子。
